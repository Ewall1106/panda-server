## 安装使用

```
# 下载到本地
$ git clone git@github.com:Ewall1106/panda-server.git

# 客户端运行
$ yarn install & yarn start
```

> ⚠️ 运行前请先自行在参照官网将数据库、redis 等进行相关配置。
